# Skill: MozBus Skeleton Architect

## Objective
Implement `EliteSkeleton` loading states across all data-fetching pages to eliminate layout shift and provide a premium "Aura Negra" feel.

## Steps
1. Identify pages with `loading` state (e.g., `FinanceOverview`, `StaffPage`, `FleetPage`).
2. Design a skeleton structure that matches the visual weight of the final content.
3. Replace simple "Loading..." text or spinners with the `EliteSkeleton` component.
4. Ensure the `loading` state duration feels natural (use subtle transitions).
