# 🌍 MozBus Global Benchmarker Skill

Esta skill especializa-se em padrões mundiais de excelência para sistemas de bilheteira (E-Ticketing) e transporte de passageiros. O seu objectivo é elevar o ecossistema MozBus ao nível das melhores plataformas globais (Ex: Trainline, FlixBus, Omio, Ticketmaster) através de avaliações rigorosas de UX, performance e funcionalidade.

## 🏆 Pilares da Elite Global (Ticketing 2026)

### 1. UX Sem Atrito (Frictionless)
*   **One-Click Booking:** Conclusão da compra em menos de 30 segundos.
*   **Checkout Minimalista:** Apenas os campos estritamente necessários.
*   **Micro-interacções:** Feedback visual instantâneo para cada acção.

### 2. Ecossistema Integrado (All-in-One)
*   **Ancillary Revenue:** Venda de serviços extra (Bagagem extra, Seguro de Viagem, Selecção de Lugar VIP) integrada no fluxo principal.
*   **Multi-modalidade:** Conexão entre diferentes tipos de transporte em uma única reserva.

### 3. Transparência Operacional
*   **Precificação Dinâmica Clara:** O utilizador entende porque o preço mudou (ex: "Apenas 3 lugares restantes neste preço").
*   **Políticas de Reembolso Automáticas:** Gestão de cancelamentos sem intervenção humana.

### 4. Acessibilidade & Performance (POUR)
*   **Acessibilidade Total:** Conformidade com WCAG (Perceptível, Operável, Compreensível e Robusto).
*   **Performance 100/100:** LCP (Largest Contentful Paint) em menos de 1.2s.

## 🔍 Protocolo de Avaliação MozBus

Para avaliar o ecossistema, analisa os seguintes itens:

1.  **Fluxo de Venda:** É possível comprar sem criar conta (Guest Checkout)? O pagamento é integrado?
2.  **Gestão de Bagagem:** O registro de carga é intuitivo ou parece um "remendo"?
3.  **Interface Terminal (POS):** O operador consegue despachar um passageiro em menos de 15 segundos?
4.  **Estética "Elite":** A interface transmite confiança e luxo (Aura Negra) ou parece um sistema genérico?

## 🚀 Checklist de Gap para a Elite
- [ ] Wallet Integration (Apple/Google Wallet)
- [ ] Notificações Push em Tempo Real (Estado do Autocarro)
- [ ] Sistema de Fidelização (Milhas/Pontos MozBus)
- [ ] Offline Mode para Terminais de Baixa Conectividade
